# Crime Intelligence Risk Analytics Dashboard

An end-to-end crime analytics project that transforms raw crime data into meaningful insights using Python, Oracle SQL, and Power BI.

The project includes data cleaning, preprocessing, and feature engineering using Python, structured data storage and analysis using Oracle Database, and an interactive Power BI dashboard for analyzing crime trends, severity levels, risk categories, hotspots, victim demographics, and case performance.

## Technologies Used

- Python
- Pandas
- Oracle Database
- SQL
- Power BI

## Project Workflow

Raw Crime Dataset  
↓  
Python Data Cleaning & Feature Engineering  
↓  
Oracle Database  
↓  
SQL Analysis  
↓  
Power BI Interactive Dashboard

## Features

- Crime trend analysis
- Crime severity analysis
- Risk category analysis
- City-wise crime hotspot visualization
- Victim demographic analysis
- Case closure analysis
